並列フィルタ¶

VTKライブラリーは並列ファイル・フォーマットをサポートしています．複数のファイルに分割されたメッシュの読み込みは，VTKおよびPyVistaでネイティブにサポートされています．

import os

# sphinx_gallery_thumbnail_number = 1
import pyvista as pv
from pyvista import examples

いくつかのファイルに分割された pyvista.UnstructuredGrid を含むサンプルデータセットをダウンロードしてみましょう．

# Do not capture output because we'll demo how to read the file

上記のコードは，血管メッシュの並列ファイルのセットを含むデータセットをダウンロードし， pyvista.UnstructuredGrid を返しました - 私たちは，これらのタイプのファイルの使用方法をデモできるように，そのUnstructuredGridを取得しませんでした．

これがデータセットをダウンロードした場所を調べてみましょう:

path = os.path.join(pv.EXAMPLES_PATH, "blood_vessels")
os.listdir(path)

出力:

['T0000000500', 'T0000000500.pvtu']

os.listdir(os.path.join(path, "T0000000500"))

出力:

['002.vtu', '000.vtu', '001.vtu', '003.vtu']

.pvtu ファイルはディレクトリの横にあるので注意してください．このディレクトリには，メッシュ全体を構成するすべての並列ファイルまたはピースが含まれています． .pvtu ファイルを読み取るだけで，VTKがメッシュをまとめます．

filename = os.path.join(path, "T0000000500.pvtu")
mesh = pv.read(filename)
