PyVistaで itkwidgets を使う¶

注釈

バージョン 0.32.0 の時点では， ipyvtk-simple はJupyterlab 3をサポートしていません．次のコマンドを実行しようとすると，jupyterlab内で Model not found エラーが返されます．

Issue 405 でこの進捗を追跡します．

PyVistaには，Jupyterでプロットを視覚化するためのインターフェースがあります． pyvista.PlotterITK クラスを使用すると，jupyterノートブック内でメッシュをインタラクティブに視覚化できます．jupyter内でのプロットを好む人にとって，これは VTK と pyvista を使用して視覚化する優れた方法です．

itkwidgets ライブラリについては @thewtex に特に感謝します．

インストール¶

PlotterITK を使用するには， itkwidgets>=0.25.2 をインストールする必要があります． ここ のインストール手順に従ってください．

もしあなたがcondaを使いたくないなら， pip で全てをインストールできますが，juptyerlabが最新であることを確認してください．問題が発生した場合は，pipを使用してjupyterlabをアンインストールして再インストールします．

ITKwidgetsを使用したプロットの例¶

次の例は，単純な球を表示する単純なプロットを作成する方法を示しています．

import pyvista as pv

# create a mesh and identify some scalars you wish to plot
mesh = pv.Sphere()
z = mesh.points[:, 2]

# Plot using the ITKplotter
pl = pv.PlotterITK()
pl.add_mesh(mesh, scalars=z, smooth_shading=True)
pl.show(True)

便宜上， plot_itk 関数を使用して図をプロットすることもできます．

import pyvista as pv

# create a mesh and identify some scalars you wish to plot
mesh = pv.Sphere()
z = mesh.points[:, 2]

# Plot using the itkwidget
pv.plot_itk(mesh, scalars=z)

class PlotterITK(**kwargs)¶

ベースクラス: object

ITKウィジェットプロッタ．

jupyterノートブック内でインタラクティブにプロットするために使用します． itkwidgets>=0.25.2 が必要です．インストールについては以下を参照してください．

https://github.com/InsightSoftwareConsortium/itkwidgets#installation

例

>>> import pyvista
>>> mesh = pyvista.Sphere()
>>> pl = pyvista.PlotterITK()  
>>> pl.add_mesh(mesh, color='w')  
>>> pl.background_color = 'k'  
>>> pl.show()
add_mesh(mesh, color=None, scalars=None, opacity=1.0, smooth_shading=False)¶

PyVista/VTKメッシュまたはデータセットを追加します．

itkwidgetsがシーンにラップできるPyVista/VTKメッシュを追加します．

パラメータ
meshpyvista.DataSet or pyvista.MultiBlock

PyVistaまたはVTKメッシュはサポートされています．また， pyvista.wrap() が扱うことができる任意のデータセットは，XYZ点のNumPy配列を含みます．

colorstr sequence, optional

メッシュ全体を単一のソリッドカラーにします．文字列，RGBリスト，または16進カラー文字列．例: color='white' ， color='w' ， color=[1, 1, 1] ， color='#FFFFFF' ．スカラーが指定されている場合，色は上書きされます．

scalarsstr or numpy.ndarray, optional

メッシュの "色" に使用されるスカラー．メッシュ上に存在する配列の文字列名，またはメッシュ内のセル数またはポイント数と等しい配列を受け入れます．配列は単一のベクトルとしてサイズを設定する必要があります． color と scalars が両方 None の場合，アクティブスカラーが使用されます．

opacityfloat, optional

Opacity of the mesh. If a single float value is given, it will be the global opacity of the mesh and uniformly applied everywhere - should be between 0 and 1. Default 1.0.

smooth_shadingbool, optional

Smooth mesh surface mesh by taking into account surface normals. Surface will appear smoother while sharp edges will still look sharp. Default False.

add_points(points, color=None, point_size=3.0)¶

プロッタに点を追加します．

パラメータ
pointsnumpy.ndarray or pyvista.DataSet

An n x 3 numpy array of points or PyVista dataset with points.

colorstr or sequence, optional

Either a string, RGB sequence, or hex color string. For one of the following.

  • color='white'

  • color='w'

  • color=[1, 1, 1]

  • color='#FFFFFF'

point_sizefloat, optional

Point size of any nodes in the dataset plotted. Also applicable when style='points'. Default 3.0.

例

プロッタに10個のランダム点を追加する

>>> add_points(np.random.random((10, 3)), 'r', 10)

show(ui_collapsed=True, rotate=False, show_bounds=False, **kwargs)¶

セル出力にitkwidgetsプロッタを表示します．

パラメータ
ui_collapsedbool, optional

ユーザインタフェースを集約してプロットします．プロット時にUIを有効にできます．デフォルトは False です．

rotatebool, optional

シーンを中心にカメラを回転します．デフォルトの False ．計算量が多いように見えます．

show_boundsbool, optional

Show the bounding box. Default False.

**kwargsdict, optional

Additional arguments to pass to itkwidgets.Viewer.
