pyvista.ParametricEllipsoid(xradius=None, yradius=None, zradius=None, **kwargs)¶

準拠楕円体を生成します．

ParametricEllipsoidは，準拠楕円体を生成します．すべての半径が同じであれば，球があります． RadiusX = RadiusY > RadiusZ の場合，偏球が発生します．ここでは，Z軸が対称軸を形成します．まず第一に，これは地球の形です． RadiusX = RadiusY < RadiusZ の場合は，長楕円体になります．

パラメータ

  • xradius (float, optional) -- X軸の倍率．デフォルトは1です．

  • yradius (float, optional) -- Y軸の倍率．デフォルトは1です．

  • zradius (float, optional) -- Z軸の倍率．デフォルトは1です．

戻り値

surf -- ParametricEllipsoid表面

戻り値の型

pyvista.PolyData

サンプル

ParametricEllipsoidメッシュを作成

>>> import pyvista
>>> mesh = pyvista.ParametricEllipsoid()
>>> cpos = mesh.plot(color='w', smooth_shading=True)

pyvista.ParametricRandomHills(numberofhills=None, hillxvariance=None, hillyvariance=None, hillamplitude=None, randomseed=None, xvariancescalefactor=None, yvariancescalefactor=None, amplitudescalefactor=None, **kwargs)¶

ランダムに配置された丘で覆われた表面を生成します．

ParametricRandomHillsは，ランダムに配置された丘で覆われた表面を生成します．丘の形状と高さは，隣接する丘の存在が丘の形状と高さに影響するため，丘の形状と高さはさまざまです．サーフェス上の通常のグリッドに丘を配置するためのオプションが用意されています．この場合，丘はすべて同じ形状と高さになります．

パラメータ

  • numberofhills (int, optional) -- 丘の数．デフォルトは30です．

  • hillxvariance (float, optional) -- x方向の丘の変化．デフォルトは2.5です．

  • hillyvariance (float, optional) -- y方向の丘の変化．デフォルトは2.5です．

  • hillamplitude (float, optional) -- 丘の振幅 (高さ) ．デフォルトは2です．

  • randomseed (int, optional) -- 乱数ジェネレータのシード値を1に設定すると乱数ジェネレータが初期化され，負の値に設定するとシステム時間で初期化されます．デフォルトは1です．

  • xvariancescalefactor (float, optional) -- x方向の分散のスケール係数．デフォルトは13です．

  • yvariancescalefactor (float, optional) -- y方向の分散のスケール係数．デフォルトは13です．

  • amplitudescalefactor (float, optional) -- 振幅のスケール係数．デフォルトは13です．

戻り値

surf -- ParametricRandomHills表面

戻り値の型

pyvista.PolyData

サンプル

ParametricRandomHillsメッシュを生成

>>> import pyvista
>>> mesh = pyvista.ParametricRandomHills()
>>> cpos = mesh.plot(color='w', smooth_shading=True)

pyvista.ParametricSuperEllipsoid(xradius=None, yradius=None, zradius=None, n1=None, n2=None, **kwargs)¶

超楕円体を生成します．

ParametricSuperEllipsoidは，超楕円体を生成します．超楕円体は，n1とn2の2つのパラメータによって制御される汎用プリミティブです．特殊な場合として，球，正方形のボックス，および閉じた円柱状の缶を表すことができます．

パラメータ

  • xradius (float, optional) -- X軸の倍率．デフォルトは1です．

  • yradius (float, optional) -- Y軸の倍率．デフォルトは1です．

  • zradius (float, optional) -- Z軸の倍率．デフォルトは1です．

  • n1 (float, optional) -- z軸の "直角度" パラメータ．デフォルトは1です．

  • n2 (float, optional) -- x-y平面の "squareness" パラメータ．デフォルトは1です．

戻り値

surf -- ParametricSuperEllipsoid表面

戻り値の型

pyvista.PolyData

サンプル

ParametricSuperEllipsoidメッシュを生成

>>> import pyvista
>>> mesh = pyvista.ParametricSuperEllipsoid()
>>> cpos = mesh.plot(color='w', smooth_shading=True)
pyvista.ParametricSuperToroid(ringradius=None, crosssectionradius=None, xradius=None, yradius=None, zradius=None, n1=None, n2=None, **kwargs)¶

スーパートロイドを生成します．

ParametricSuperToroidがスーパートロイドを生成します．本質的にスーパートロイドは，正弦項と余弦項を累乗したトーラスです．スーパートロイドは，4つのパラメータr0，r1，n1，およびn2によって制御される汎用プリミティブです．r0，r 1はトーラスのタイプを決定し，n1の値はトーラスリングの形状を決定し，n2はリングの断面の形状を決定します．これらのパワーの異なる値によって，形状がすべて基本的にトロイダルな3 D形状のファミリが生成されます．

パラメータ

  • ringradius (float, optional) -- スーパートロイドのリングの中心から中心までの半径．デフォルトは1です．

  • crosssectionradius (float, optional) -- スーパートロイドのリングの断面の半径．デフォルトは0.5です．

  • xradius (float, optional) -- X軸の倍率．デフォルトは1です．

  • yradius (float, optional) -- Y軸の倍率．デフォルトは1です．

  • zradius (float, optional) -- Z軸の倍率．デフォルトは1です．

  • n1 (float, optional) -- トーラスリングの形状．デフォルトは1です．

  • n2 (float, optional) -- リングの断面の形状．デフォルトは1です．

戻り値

surf -- ParametricSuperToroid表面

戻り値の型

pyvista.PolyData

サンプル

ParametricSuperToroidメッシュを生成

>>> import pyvista
>>> mesh = pyvista.ParametricSuperToroid()
>>> cpos = mesh.plot(color='w', smooth_shading=True)
