ジオメトリオブジェクト¶

PyVistaには，単純なジオメトリオブジェクトを生成する関数がいくつかあります．これらの関数のコードドキュメントは，このページにあります．これらの関数が何を生成するかを調べるには，例を調べてください : ジオメトリオブジェクト ．

pyvista.Sphere(radius=0.5, center=(0, 0, 0), direction=(0, 0, 1), theta_resolution=30, phi_resolution=30, start_theta=0, end_theta=360, start_phi=0, end_phi=180)¶

vtk球を作成します．

パラメータ

  • radius (float, optional) -- 球の半径

  • center (np.ndarray or list, optional) -- [x, y, z] の中心

  • direction (list or tuple or np.ndarray) -- 球の上端が [x, y, z] 内を指す方向

  • theta_resolution (int , optional) -- 経度方向のポイント数を設定します(start_thetaからend thetaまでの範囲)．

  • phi_resolution (int, optional) -- 緯度方向のポイント数を設定します(start_phiからend_phiまで)．

  • start_theta (float, optional) -- 開始経度角度．

  • end_theta (float, optional) -- 終了経度角度．

  • start_phi (float, optional) -- 開始緯度角度．

  • end_phi (float, optional) -- 終了緯度角度．

戻り値

sphere -- 球メッシュ．

戻り値の型

pyvista.PolyData

pyvista.Cylinder(center=(0.0, 0.0, 0.0), direction=(1.0, 0.0, 0.0), radius=0.5, height=1.0, resolution=100, capping=True, **kwargs)¶

円柱の表面を作成します．

pyvista.CylinderStructured() も参照してください．

パラメータ

  • center (list or tuple or np.ndarray) -- [x, y, z] 中のセントロイドの位置

  • direction (list or tuple or np.ndarray) -- [x, y, z] 内の方向円柱

  • radius (float) -- 円柱の半径．

  • height (float) -- 円柱の高さ．

  • resolution (int) -- 円柱の円形面上の点の数です．

  • capping (bool, optional) -- キャップ円柱の終点はポリゴンです．デフォルトはTrue

戻り値

cylinder -- 円柱表面．

戻り値の型

pyvista.PolyData

サンプル

>>> import pyvista
>>> import numpy as np
>>> cylinder = pyvista.Cylinder(np.array([1, 2, 3]), np.array([1, 1, 1]), 1, 1)
>>> cpos = cylinder.plot()
pyvista.CylinderStructured(radius=0.5, height=1.0, center=(0.0, 0.0, 0.0), direction=(1.0, 0.0, 0.0), theta_resolution=32, z_resolution=10)¶

pyvista.StructuredGrid として円柱メッシュを作成します．

エンドキャップは開いたままです．これにより， radius に単一の値が指定された場合は表面メッシュが作成され， radius 引数に複数の半径がリスト/配列として指定された場合は3 Dメッシュが作成されます．

パラメータ

  • radius (float) -- 円柱の半径．繰り返し可能

  • height (float) -- Z軸に沿った円柱の高さ (長さ)

  • center (list or tuple or np.ndarray) -- [x, y, z] 中のセントロイドの位置

  • direction (list or tuple or np.ndarray) -- [x, y, z] の方向円柱Z軸

  • theta_resolution (int) -- 円柱の円形面上の点の数です．

  • z_resolution (int) -- 円柱の高さ (Z軸) に沿った点の数

pyvista.Cube(center=(0.0, 0.0, 0.0), x_length=1.0, y_length=1.0, z_length=1.0, bounds=None)¶

  • y_length (float) -- y方向の立方体の長さ．

  • z_length (float) -- z方向の立方体の長さ．

  • bounds (np.ndarray or list) -- 立方体のバウンディングボックスを指定します．指定した場合，他のすべての引数は無視されます． (xMin,xMax, yMin,yMax, zMin,zMax)

pyvista.Spline(points, n_points=None)¶

点からスプラインを作成します．

パラメータ

  • points (np.ndarray) -- スプラインを作成するポイントの配列です．配列は3 Dで方向が順序付けられている必要があります．

  • n_points (int, optional) -- 点配列に沿って補間する点の数．

戻り値

spline -- スプラインのラインメッシュです．

戻り値の型

pyvista.PolyData

サンプル

スプラインを作成します

>>> import numpy as np
>>> import pyvista as pv
>>> theta = np.linspace(-4 * np.pi, 4 * np.pi, 100)
>>> z = np.linspace(-2, 2, 100)
>>> r = z**2 + 1
>>> x = r * np.sin(theta)
>>> y = r * np.cos(theta)
>>> points = np.column_stack((x, y, z))
>>> spline = pv.Spline(points, 1000)

pyvista.CircularArc(pointa, pointb, center, resolution=100, negative=False)¶

2つの端点と中心で定義される円弧を作成します．

ポリラインを構成するセグメントの数は，オブジェクトの解像度を設定することによってコントロールされます．

パラメータ

  • pointa (np.ndarray or list) -- 最初の終点の位置です．

  • pointb (np.ndarray or list) -- もう一方の終点の位置．

  • center (np.ndarray or list) -- 円弧を定義する円の中心．

  • resolution (int, optional) -- 円弧を描くポリラインのセグメント数．解像度を1にすると，直線が作成されます．

  • negative (bool, optional) -- デフォルトでは，円弧は pointa と pointb の間の最短の角度セクタをスパンします．これを True に設定すると，最長の角度セクタが代わりに使用されます(すなわち，最小値1に対する負の同極角)．

サンプル

XY平面の原点を中心とする4分円弧を作成します．

>>> import pyvista
>>> arc = pyvista.CircularArc([-1, 0, 0], [0, 1, 0], [0, 0, 0])
>>> pl = pyvista.Plotter()
>>> _ = pl.add_mesh(arc, color='k', line_width=4)
>>> _ = pl.show_bounds(location='all')
>>> _ = pl.view_xy()
>>> cpos = pl.show()
pyvista.CircularArcFromNormal(center, resolution=100, normal=None, polar=None, angle=None)¶

円弧の平面に垂直に定義される円弧と角度を作成します．

ポリラインを構成するセグメントの数は，オブジェクトの解像度を設定することによってコントロールされます．

パラメータ

  • center (np.ndarray or list) -- 円弧を定義する円の中心．

  • resolution (int, optional) -- 円弧を描くポリラインのセグメント数．解像度を1にすると，直線が作成されます．

  • normal (np.ndarray or list, optional) -- 円弧の平面に対する法線ベクトルです．既定では，正のZ方向を指します．

  • polar (np.ndarray or list, optional) -- 極座標での円弧の始点．既定では，正のx方向の単位ベクトルです．

  • angle (float, optional) -- 極ベクトルを始点とする円弧の長さ(度単位)方向は反時計回りです．デフォルトは90です．

サンプル

XY平面の原点を中心とする4分円弧．

>>> import pyvista
>>> normal = [0, 0, 1]
>>> polar = [-1, 0, 0]
>>> arc = pyvista.CircularArcFromNormal([0, 0, 0], normal=normal, polar=polar)
>>> pl = pyvista.Plotter()
>>> _ = pl.add_mesh(arc, color='k', line_width=4)
>>> _ = pl.show_bounds(location='all')
>>> _ = pl.view_xy()
>>> cpos = pl.show()
pyvista.Pyramid(points)¶

5点で定義される角錐を作成します．

パラメータ

points (np.ndarray or list) -- 角錐の点．最初の4つの点が四角形の面上の反時計回りの4つの点になり，最後の点が頂点になるように，点が順序付けられます．

戻り値

pyramid

戻り値の型

pyvista.UnstructuredGrid

サンプル

>>> import pyvista
>>> pointa = [1.0, 1.0, 1.0]
>>> pointb = [-1.0, 1.0, 1.0]
>>> pointc = [-1.0, -1.0, 1.0]
>>> pointd = [1.0, -1.0, 1.0]
>>> pointe = [0.0, 0.0, 0.0]
>>> pyramid = pyvista.Pyramid([pointa, pointb, pointc, pointd, pointe])
>>> cpos = pyramid.plot()
