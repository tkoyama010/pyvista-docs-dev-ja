pyvista.TimeReader¶

class TimeReader¶

時間をサポートするリーダーのための抽象クラスです．

メソッド

TimeReader.set_active_time_point(time_point)

アクティブ時間やイテレーションをインデックスで設定します．

TimeReader.set_active_time_value(time_value)

アクティブ時刻またはイテレーション値を設定します．

TimeReader.time_point_value(time_point)

インデックスによるタイムポイントまたはイテレーションの値．

アトリビュート

TimeReader.active_time_value

アクティブタイムまたはイテレーション値．

TimeReader.number_time_points

読み取り可能なタイムポイントまたはイタレーションの数を返します．

TimeReader.time_values

全タイムまたはイテレーション値．
