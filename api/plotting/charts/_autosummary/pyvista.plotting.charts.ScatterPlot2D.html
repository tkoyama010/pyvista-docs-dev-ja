pyvista.plotting.charts.ScatterPlot2D¶

class ScatterPlot2D(x, y, color='b', size=10, style='o', label='')¶

2Dの散布図を表すクラス．

ユーザーは通常，新しいプロットインスタンスを直接作成せず，専用の2Dチャートのプロットメソッドを使用する必要があります．

パラメータ
xarray_like

描画する点のX座標．

yarray_like

描画する点のY座標．

colorcolor, オプション

このプロットに描かれる点の色です． pyvista.parse_color で解析可能な任意の色が使用できます．デフォルトは "b" です．

sizefloat, optional

このプロットに描かれるポイントマーカーのサイズです．デフォルトは 10 です．

stylestr, optional

このプロットに描画されるポイントマーカのスタイルです．許可されるマーカースタイルのリストは ScatterPlot2D.MARKER_STYLES を参照してください．デフォルトは "o" です．

labelstr, optional

このプロットのラベルで，チャートの凡例に表示されます．デフォルトは "" です．

備考

MARKER_STYLESdict

許可されたすべてのマーカースタイルをキーとする辞書．

例

単純な正弦波を散布図にプロットします．

>>> import pyvista
>>> import numpy as np
>>> x = np.linspace(0, 2*np.pi, 20)
>>> y = np.sin(x)
>>> chart = pyvista.Chart2D()
>>> _ = chart.scatter(x, y)
>>> chart.show()

プロットの表示を切り替えます．

ScatterPlot2D.update(x, y)

このプロットのポイントを更新します．

アトリビュート

ScatterPlot2D.brush

Brushオブジェクトは，このプロットの形状がどのように塗りつぶされるかを制御します．

ScatterPlot2D.color

プロットの色を返したり設定したりします．

ScatterPlot2D.label

チャートの凡例に表示されている，このプロットのラベルを返すか設定します．

ScatterPlot2D.line_style

このプロットに描かれたすべての線のラインスタイルを返すか設定します．

ScatterPlot2D.line_width

このプロットに描かれたすべての線のライン幅を返すか設定します．

ScatterPlot2D.marker_size

プロットのマーカーサイズを返すか設定します．

ScatterPlot2D.marker_style

プロットのマーカースタイルを返す，または設定します．

ScatterPlot2D.pen

Penオブジェクトは，このプロットの線がどのように描かれるかを制御します．

ScatterPlot2D.visible

図の見え方を返すか設定します．

ScatterPlot2D.x

このプロットの点のX座標を取得します．

ScatterPlot2D.y

このプロットの点のY座標を取得します．
