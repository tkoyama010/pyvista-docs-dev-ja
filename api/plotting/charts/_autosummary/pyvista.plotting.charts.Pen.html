pyvista.plotting.charts.Pen¶

class Pen(color='k', width=1, style='-', *, _wrap=None)¶

VTK PenのPythonicなラッパーで，線を描くのに使われます．

パラメータ
colorcolor, オプション

このペンで描いた線の色です． pyvista.parse_color() で解析可能な任意の色が使用できます．デフォルトは "k" です．

widthfloat, optional

このペンで描いた線の幅です．デフォルトは 1 です．

stylestr, optional

このペンで描いた線のスタイルです．使用可能なラインスタイルのリストは Pen.LINE_STYLES を参照してください．デフォルトは "-" です．

その他のパラメータ
_wrapvtk.vtkPen, optional

既存のVTK Penインスタンスをラップします．デフォルトは None (ラップしない)です．

備考

LINE_STYLESdict

許可されたすべての線スタイルをキーとする辞書．
