label¶

property ScatterPlot2D.label¶

チャートの凡例に表示されている，このプロットのラベルを返すか設定します．

例

カスタムラベル付きの2次元散布図を作成します．

>>> import pyvista
>>> chart = pyvista.Chart2D()
>>> plot = chart.scatter([0, 1, 2, 3, 4], [2, 1, 3, 4, 2])
>>> plot.label = "My awesome plot"
>>> chart.show()
