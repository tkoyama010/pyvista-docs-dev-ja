pyvista.plotting.charts.PiePlot¶

class PiePlot(data, colors=None, labels=None, *, _wrap=None)¶

円グラフを表すクラス．

ユーザーは通常，新しいプロットインスタンスを直接作成せず，専用の ChartPie クラスを使用する必要があります．

パラメータ
dataarray_like

各円セグメントの相対的な大きさ

colors : list か tuple of color, optionalpython:list または color の python:tuple, optional

このプロットに描かれるセグメントの色です． pyvista.parse_color で解析可能な任意の色が使用できます．デフォルトは None です．

labels : list か tuple of str, optionalpython:list または python:str の python:tuple, optional

チャートの凡例に表示される，描画された各パイセグメントのラベルです．デフォルトでは [] です．

その他のパラメータ
_wrapvtk.vtkPlotPie, optional

既存のVTK PlotPieインスタンスをラップします．デフォルトは None (ラップしない)です．

例

税金の使い道を示す円グラフを作成します．

>>> import pyvista
>>> x = [128.3, 32.9, 31.8, 29.3, 21.2]
>>> l = ["Social benefits", "Governance", "Economic policy", "Education", "Other"]
>>> chart = pyvista.ChartPie(x, labels=l)
>>> chart.show()

PiePlot.update(data)
