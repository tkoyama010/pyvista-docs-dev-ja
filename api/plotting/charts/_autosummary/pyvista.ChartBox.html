pyvista.ChartBox¶

class ChartBox(data, colors=None, labels=None)¶

箱ひげ図を描くための専用チャート．

パラメータ
data : list か tuple of array_likepython:list または python:tuple または numpy:array_like

(複数の)箱ひげ図を描くために関連する統計を計算するデータセット．

colors : list か tuple of color, optionalpython:list または color の python:tuple, optional

描画される箱ひげ図に使われる色です．デフォルトは None で，これはデフォルトのカラースキームを使用します．

labels : list か tuple of str, optionalpython:list または python:str の python:tuple, optional

チャートの凡例に表示される，描画された各ボックスプロットのラベルです．デフォルトでは [] です．

例

シフトした正規分布からサンプリングしたデータセットの箱ひげ図を作成します．

>>> import pyvista
>>> import numpy as np
>>> rng = np.random.default_rng(1)  # Seeded random number generator used for data generation
>>> normal_data = [rng.normal(i, size=50) for i in range(5)]
>>> chart = pyvista.ChartBox(normal_data, labels=[f"x ~ N({i},1)" for i in range(5)])
>>> chart.show()

メソッド

ChartBox.show([off_screen, full_screen, ...])

このチャートを自作のプロッターに表示します．

ChartBox.toggle()

チャートの表示を切り替えます．
