title¶

property ChartMPL.title¶

チャートのタイトルを返すか設定します。

例

'My Chart' というタイトルでmatplotlibチャートを作成します。

>>> import pyvista
>>> import matplotlib.pyplot as plt
>>> f, ax = plt.subplots()
>>> _ = ax.plot([0, 1, 2], [2, 1, 3])

>>> chart = pyvista.ChartMPL(f)
>>> chart.title = 'My Chart'
>>> chart.show()
