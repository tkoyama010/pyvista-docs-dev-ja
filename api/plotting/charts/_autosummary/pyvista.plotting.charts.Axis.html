pyvista.plotting.charts.Axis¶

class Axis(label='', range=None, grid=True, *, _wrap=None)¶

2Dチャートで使用されるVTK AxisのPythonicインターフェースです．

パラメータ
labelstr, optional

軸のラベル．デフォルトでは空の文字列 "" （ラベルが表示されない）が設定されます．

range : list か tuple of float, optionalpython:list または python:float, optional の python:tuple

軸の範囲，この軸に表示される最小値と最大値を指定します．これを None 以外の有効な値に設定すると，この軸の動作は 'fixed' に変更されます．これを None に設定すると，この軸の動作を 'auto' に変更します．デフォルトは None (軸を自動的にスケールする)です．

gridbool, optional

この軸のグリッド線の表示/非表示を切り替えるフラグです．デフォルトは True です．

その他のパラメータ
_wrapvtk.vtkAxis, optional

既存のVTK Axisインスタンスをラップします．デフォルトは None (ラップしない)です．

アトリビュート
penPen

軸の描画に使用するペン．

grid_penPen

グリッド線の描画に使用するペン．

メソッド

Axis.toggle()

軸の表示を切り替えます．
