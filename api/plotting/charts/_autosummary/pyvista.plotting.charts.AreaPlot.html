pyvista.plotting.charts.AreaPlot¶

class AreaPlot(x, y1, y2=None, color='b', label='')¶

2Dの面グラフを表すクラス．

ユーザーは通常，新しいプロットインスタンスを直接作成せず，専用の2Dチャートのプロットメソッドを使用する必要があります．

パラメータ
xarray_like

描画する領域を囲む点のX座標．

y1array_like

描画する領域の最初の輪郭上の点のY座標．

y2array_like, optional

描画する領域の2番目の輪郭上の点のY座標．デフォルトでは，ゼロの連続です．

colorcolor, オプション

このプロットに描かれる領域の色です． pyvista.parse_color で解析可能な任意の色が使用できます．デフォルトは "b" です．

labelstr, optional

このプロットのラベルで，チャートの凡例に表示されます．デフォルトは "" です．

例

各月に観測された降水量の最小値と最大値を面グラフにしてみましょう．

>>> import pyvista
>>> import numpy as np
>>> x = np.arange(12)
>>> p_min = [11, 0, 16, 2, 23, 18, 25, 17, 9, 12, 14, 21]
>>> p_max = [87, 64, 92, 73, 91, 94, 107, 101, 84, 88, 95, 103]
>>> chart = pyvista.Chart2D()
>>> _ = chart.area(x, p_min, p_max)
>>> chart.x_axis.tick_locations = x
>>> chart.x_axis.tick_labels = ["Jan", "Feb", "Mar", "Apr", "May",
...                             "Jun", "Jul", "Aug", "Sep", "Oct",
...                             "Nov", "Dec"]
>>> chart.x_axis.label = "Month"
>>> chart.y_axis.label = "Precipitation [mm]"
>>> chart.show()

メソッド

AreaPlot.toggle()

プロットの表示を切り替えます．

AreaPlot.update(x, y1[, y2])

このプロットのポイントを更新し，描画する領域の輪郭を描きます．

アトリビュート

AreaPlot.brush

Brushオブジェクトは，このプロットの形状がどのように塗りつぶされるかを制御します．

AreaPlot.color

プロットの色を返したり設定したりします．

AreaPlot.label

チャートの凡例に表示されている，このプロットのラベルを返すか設定します．

AreaPlot.line_style

このプロットに描かれたすべての線のラインスタイルを返すか設定します．

AreaPlot.line_width

このプロットに描かれたすべての線のライン幅を返すか設定します．

AreaPlot.pen

Penオブジェクトは，このプロットの線がどのように描かれるかを制御します．

AreaPlot.visible

図の見え方を返すか設定します．

AreaPlot.x

描画領域を囲む点のX座標を取得します．

AreaPlot.y1

描画領域の最初の輪郭上の点のY座標を取得します．

AreaPlot.y2

描画領域の第2の輪郭上の点のY座標を取得します．
