create_axes_orientation_box¶

create_axes_orientation_box(line_width=1, text_scale=0.366667, edge_color='black', x_color=None, y_color=None, z_color=None, xlabel='X', ylabel='Y', zlabel='Z', x_face_color='red', y_face_color='green', z_face_color='blue', color_box=False, label_color=None, labels_off=False, opacity=0.5)¶

ラベル付きボックス軸方向ウィジェットを作成します．

パラメータ
line_widthfloat, optional

マーカーラインの幅．

text_scalefloat, optional

顔に対するテキストのサイズ．

edge_colorstr or sequence, optional

エッジの色．

x_colorstr or sequence, optional

X軸テキストの色．

y_colorstr or sequence, optional

Y軸テキストの色．

z_colorstr or sequence, optional

Z軸テキストの色．

xlabelstr, optional

x軸に使用するテキスト．

ylabelstr, optional

y軸に使用するテキスト．

zlabelstr, optional

z軸に使用するテキスト．

x_face_colorstr or sequence, optional

x軸の矢印に使用する色． デフォルトはテーマ軸パラメータ．

y_face_colorstr or sequence, optional

y軸の矢印に使用する色． デフォルトはテーマ軸パラメータ．

z_face_colorstr or sequence, optional

z軸の矢印に使用する色． デフォルトはテーマ軸パラメータ．

color_boxbool, optional

面の色を有効または無効にします． それ以外の場合，ボックスは白になります．

label_colorstr or sequence, optional

ラベルの色．

labels_offbool, optional

軸のテキストラベルを有効または無効にします．

opacityfloat, optional

オリエンテーションボックスの不透明度を [0, 1] の範囲で指定します．

戻り値
vtk.vtkAnnotatedCubeActor

注釈付きの立方体のアクター．

例

オリエンテーションボックスの作成とプロッティング

>>> import pyvista
>>> actor = pyvista.create_axes_orientation_box(
...    line_width=1, text_scale=0.53,
...    edge_color='black', x_color='k',
...    y_color=None, z_color=None,
...    xlabel='X', ylabel='Y', zlabel='Z',
...    color_box=False,
...    labels_off=False, opacity=1.0)
>>> pl = pyvista.Plotter()
>>> _ = pl.add_actor(actor)
>>> pl.show()
