pyvista.themes._Font¶

class _Font[ソース]¶

PyVistaのプロッターフォント設定．

例

既定のフォントファミリを 'arial' に設定します． 'arial' ， 'courier' ，または 'times' のいずれかである必要があります．

>>> import pyvista
>>> pyvista.global_theme.font.family = 'arial'

デフォルトのフォントサイズを20に設定します．

>>> pyvista.global_theme.font.size = 20

デフォルトのタイトルサイズを40に設定します

>>> pyvista.global_theme.font.title_size = 40

デフォルトのラベルサイズを10に設定します

>>> pyvista.global_theme.font.label_size = 10

デフォルトのテキストの色を 'grey' に設定します

>>> pyvista.global_theme.font.color = 'grey'

数値データのフォーマットに使われる文字列フォーマッタを '%.6e' に設定します

>>> pyvista.global_theme.font.fmt = '%.6e'

_Font.to_dict()
