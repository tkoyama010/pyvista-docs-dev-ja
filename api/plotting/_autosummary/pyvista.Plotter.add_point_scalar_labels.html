add_point_scalar_labels¶

Plotter.add_point_scalar_labels(points, labels, fmt=None, preamble='', **kwargs)¶

データセットのポイントに，スカラー値のラベルを付けます．

pyvista.BasePlotter.add_point_labels() 用のラッパーです．

パラメータ
pointsnumpy.ndarray or pyvista.DataSet

An n x 3 numpy.ndarray or pyvista dataset with points.

labelsstr, optional

使用するポイントデータ配列の文字列名．

fmtstr, optional

数値データのフォーマットに使用する文字列フォーマッタ．

preamblestr, optional

Text before the start of each label.

**kwargsdict, optional

Keyword arguments passed to pyvista.BasePlotter.add_point_labels().

戻り値
vtk.vtkActor2D

VTK label actor. Can be used to change properties of the labels.
