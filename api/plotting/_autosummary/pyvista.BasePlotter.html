pyvista.BasePlotter¶

class pyvista.BasePlotter(*args, **kwargs)¶

プロッタおよびpyvistaqtで使用されます．QtInteractorクラス．

パラメータ

  • shape (list or tuple, optional) -- メインウィンドウ内のサブレンダリングウィンドウの数． shape=(2, 1) で2 acrossを， shape=(2, 2) で2 x 2 グリッドを指定します．デフォルトでは，レンダラは1つだけです．文字列記述子をshapeとして受け入れることもできます．例: * shape="3|1" は左側に3プロット，右側に1プロット， * shape="4/2" は上部に4プロット，下部に2プロットを意味します．

  • border (bool, optional) -- 各レンダリングウィンドウの周囲に境界を描きます．デフォルトは False です．

  • border_color (string or 3 item list, optional) -- 文字列，RGBリスト，または16進カラー文字列．例 * color='white' * color='w' * color=[1, 1, 1] * color='#FFFFFF'

  • border_width (float, optional) -- 有効になっている場合のピクセル単位のボーダーの幅 ．

  • title (str, optional) -- スカラーバーのウィンドウタイトル

  • lighting (str, optional) -- プロッタに設定する照明．使用可能なオプション: * 'light_kit': 5つのライトで構成されるvtkライトキット． * 'three lights': 3つのライトを使用したイルミネーション． * 'none': インスタンス化時に光源がありません．デフォルトはライトキット(正確には，ライトキットのように動作する5つの独立したライト)です．

  • theme (pyvista.themes.DefaultTheme, optional) -- Plot固有のテーマ．
