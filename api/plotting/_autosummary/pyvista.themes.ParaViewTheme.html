pyvista.themes.ParaViewTheme¶

class ParaViewTheme[ソース]¶

Paraviewらしいテーマです．

例

paraviewのようなテーマをグローバルデフォルトにします．

>>> import pyvista
>>> from pyvista import themes
>>> pyvista.set_plot_theme(themes.ParaViewTheme())

または，文字列で設定します．

>>> pyvista.set_plot_theme('paraview')

メソッド

ParaViewTheme.from_dict(dict_)

辞書から作成します．

ParaViewTheme.load_theme(theme)

現在のテーマをテーマで上書きします．

ParaViewTheme.restore_defaults()

テーマをデフォルトに戻します．

ParaViewTheme.save(filename)

このテーマをjsonファイルにシリアライズします．

ParaViewTheme.to_dict()

テーマの設定パラメータをディクショナリーとして返します．
