pyvista.themes.DocumentTheme¶

class DocumentTheme¶

論文やプレゼンテーションに適したドキュメントテーマです．

このテーマでは

  • 白背景

  • 黒字フォント

  • "viridis" カラーマップ

  • サーフェスプロットのエッジを無効

  • 隠れたエッジの除去

プレゼンや論文などに最適です．

例

ドキュメントテーマをグローバルデフォルトにします．

>>> import pyvista
>>> from pyvista import themes
>>> pyvista.set_plot_theme(themes.DocumentTheme())

または，文字列で設定します．

>>> pyvista.set_plot_theme('document')

メソッド

DocumentTheme.from_dict(dict_)

辞書から作成します．

DocumentTheme.load_theme(theme)

現在のテーマをテーマで上書きします．

DocumentTheme.restore_defaults()

テーマをデフォルトに戻します．

DocumentTheme.save(filename)

このテーマをjsonファイルにシリアライズします．

DocumentTheme.to_dict()

テーマの設定パラメータをディクショナリーとして返します．
