plot_arrows¶

plot_arrows(cent, direction, **kwargs)¶

矢印をベクトルとしてプロットします．

パラメータ
centnumpy.ndarray

単一の3 Dポイントまたは3 Dポイントの配列を受け入れます．

directionnumpy.ndarray

単一の3 D点または3 Dベクトルの配列を受け入れます． cent と同じ数のアイテムを含める必要があります．

**kwargsdict, optional

pyvista.plot() 参照．

戻り値
tuple

pyvista.plot() のリターンを参照してください．

参考

pyvista.plot()

例

1つのランダムな矢印をプロットします．

>>> import numpy as np
>>> import pyvista
>>> cent = np.random.random(3)
>>> direction = np.random.random(3)
>>> pyvista.plot_arrows(cent, direction)

100のランダムな矢印をプロットします．

>>> import numpy as np
>>> import pyvista
>>> cent = np.random.random((100, 3))
>>> direction = np.random.random((100, 3))
>>> pyvista.plot_arrows(cent, direction)
