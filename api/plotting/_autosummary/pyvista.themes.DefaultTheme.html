pyvista.themes.DefaultTheme¶

class DefaultTheme[ソース]¶

PyVistaのデフォルトテーマ．

例

グローバルな既定の背景色を白に変更します．

>>> import pyvista
>>> pyvista.global_theme.color = 'white'

デフォルトでエッジを表示します．

>>> pyvista.global_theme.show_edges = True

デフォルトのテーマから新しいテーマを作成し，グローバルに適用します．

>>> my_theme = pyvista.themes.DefaultTheme()
>>> my_theme.color = 'red'
>>> my_theme.background = 'white'
>>> pyvista.global_theme.load_theme(my_theme)

メソッド

DefaultTheme.from_dict(dict_)

辞書から作成します．

DefaultTheme.load_theme(theme)

現在のテーマをテーマで上書きします．

DefaultTheme.restore_defaults()

テーマをデフォルトに戻します．

DefaultTheme.save(filename)

このテーマをjsonファイルにシリアライズします．

DefaultTheme.to_dict()

テーマの設定パラメータをディクショナリーとして返します．
