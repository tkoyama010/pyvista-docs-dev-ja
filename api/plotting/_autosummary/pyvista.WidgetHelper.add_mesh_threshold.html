add_mesh_threshold¶

WidgetHelper.add_mesh_threshold(mesh, scalars=None, invert=False, widget_color=None, preference='cell', title=None, pointa=(0.4, 0.9), pointb=(0.9, 0.9), continuous=False, **kwargs)¶

スライダを使用してメッシュに閾値を適用します．

メッシュの閾値をインタラクティブに設定するために使用するスライダウィジェットを使用して，メッシュをシーンに追加します．

閾値メッシュは，プロッタの .threshold_meshes 属性に保存されます．

パラメータ
meshpyvista.DataSet

シーンおよび閾値に追加する入力データセット．

scalarsstr, optional

閾値および表示するメッシュ上のスカラーの文字列名．

invertbool, optional

閾値を反転（フリップ）させる．

widget_colorstr or sequence, optional

ウィジェットの色．文字列，RGB配列，または16進数の色文字列のいずれかです． 例えば

  • color='white'

  • color='w'

  • color=[1, 1, 1]

  • color='#FFFFFF'

preferencestr, optional

mesh.n_points == mesh.n_cells でスカラを設定した場合，このパラメータはスカラがどのようにメッシュにマッピングされるかを設定します． デフォルトの 'points' では，スカラーはメッシュのポイントに関連付けられます． これは 'points' または 'cell' のいずれかです．

titlestr, optional

スライダウィジェットの文字列ラベル．

pointasequence, optional

表示ポート上のスライダの左端の相対座標．

pointbsequence, optional

表示ポート上のスライダの右ポイントの相対座標．

continuousbool, optional

この機能が有効な場合（デフォルトは False ），閾値境界と交差するために，頂点からの離散的なスカラー値の集合ではなく，連続的な区間 [minimum cell scalar, maxmimum cell scalar] を使用します．

**kwargsdict, optional

追加のキーワード引数はすべて add_mesh に渡され，メッシュの表示方法を制御します．

戻り値
vtk.vtkActor

メッシュのVTKアクター．
