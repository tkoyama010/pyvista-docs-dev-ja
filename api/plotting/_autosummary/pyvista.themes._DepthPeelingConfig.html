pyvista.themes._DepthPeelingConfig¶

class _DepthPeelingConfig¶

PyVistaのデプスピーリングの設定．

例

グローバルなDepth Peelingのパラメータを設定します．

>>> import pyvista
>>> pyvista.global_theme.depth_peeling.number_of_peels = 1  
>>> pyvista.global_theme.depth_peeling.occlusion_ratio = 0.0  
>>> pyvista.global_theme.depth_peeling.enabled = True

メソッド

_DepthPeelingConfig.from_dict(dict_)

辞書から作成します．

_DepthPeelingConfig.to_dict()

テーマの設定パラメータをディクショナリーとして返します．

アトリビュート

_DepthPeelingConfig.enabled

Depth Peelingが有効であるかどうかを返す，または設定します．

_DepthPeelingConfig.number_of_peels

Depth Peelingのpeeling回数を返すか設定します．

_DepthPeelingConfig.occlusion_ratio

Depth Peeling時のocclusion比率を返す，または設定します．
