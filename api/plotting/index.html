チャートAPI#

Charts APIのリファレンスです．これらの専用クラスは，プロットウィンドウにチャートを埋め込むために使用することができます．なお，チャートを使用するには，VTKのバージョンが9.0以上，将来のリリースでは vtk>=9.2 が必要になります．

ウィジェットAPI#

pyvista.Plotter クラスは pyvista.WidgetHelper のすべてのウィジェットメソッドを継承するので，以下のメソッドはすべてPyVistaプロッタから利用できます．

WidgetHelper(*args, **kwargs)

ウィジェットを管理する内部クラス．
