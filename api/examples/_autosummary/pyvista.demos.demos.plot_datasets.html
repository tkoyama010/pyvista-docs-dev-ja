plot_datasets¶

plot_datasets(dataset_type=None)[ソース]¶

pyvistaのデータセットタイプをプロットします．

このデモでは，以下のPyVistaデータセットタイプをプロットします．

  • pyvista.PolyData

  • pyvista.UnstructuredGrid

  • pyvista.UniformGrid

  • pyvista.RectilinearGrid

  • pyvista.StructuredGrid

パラメータ
dataset_typestr, optional

設定されていれば，そのデータセットだけをプロットする． 以下のいずれかでなければなりません．

  • 'PolyData'

  • 'UnstructuredGrid'

  • 'UniformGrid'

  • 'RectilinearGrid'

  • 'StructuredGrid'

例

>>> from pyvista import demos
>>> demos.plot_datasets()
