pyvista.examples.downloads¶

様々なソースから集められたダウンロード可能なデータセット．

一度ダウンロードしたデータセットはローカルに保存されるため，迅速に再利用することができます．

例¶

>>> from pyvista import examples
>>> mesh = examples.download_saddle_surface()
>>> mesh.plot()
