orientation_plotter()¶

オリエンテーションキューブを含むプロッタを返します．

戻り値
pyvista.Plotter

回転した立方体のプロッター．

例

>>> from pyvista import demos
>>> plotter = demos.orientation_plotter()
>>> plotter.show()
