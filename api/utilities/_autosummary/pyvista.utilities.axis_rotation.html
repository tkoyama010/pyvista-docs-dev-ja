axis_rotation¶

axis_rotation(points, angle, inplace=False, deg=True, axis='z')[ソース]¶

軸を中心に角度をつけて点を回転させます．

パラメータ
pointsnumpy.ndarray

形状が (N, 3) である点の配列です．

anglefloat

回転角度．

inplacebool, optional

何も返さずにin-placeポイントを更新します．

degbool, optional

True の場合，角度はラジアンではなく，度として解釈されます．デフォルトは True です．

axisstr, optional

回転させる軸の名前を指定します．有効なオプションは 'x', 'y', および 'z' です．デフォルトの値は 'z' です．

戻り値
numpy.ndarray

回転した点．

例

点の集合をX軸を中心に90度回転させるインプレース機能．

>>> import numpy as np
>>> import pyvista
>>> from pyvista import examples
>>> points = examples.load_airplane().points
>>> points_orig = points.copy()
>>> pyvista.axis_rotation(points, 90, axis='x', deg=True, inplace=True)
>>> assert np.all(np.isclose(points[:, 0], points_orig[:, 0]))
>>> assert np.all(np.isclose(points[:, 1], -points_orig[:, 2]))
>>> assert np.all(np.isclose(points[:, 2], points_orig[:, 1]))
