is_pyvista_dataset¶

is_pyvista_dataset(obj)¶

ObjectがPyVistaでラップされたデータセットである場合， True を取得します．

パラメータ
objanything

テストする任意のオブジェクト．

戻り値
bool

オブジェクトが pyvista.DataSet の場合は True となります．
