parse_color¶

parse_color(color, opacity=None, default_color=None)[ソース]¶

色をVTKフレンドリーなRGB(A)タプルに解析します．

バージョン 0.34.0 で非推奨: 代わりに Color を使用して、色をパースして変換します。

color が RGBA の浮動小数点数のシーケンスである場合，opacity パラメータは無視されます．

返される値は0から1の間です．

パラメータ
colorcolor_like

文字列，RGB配列，RGBA配列，または16進数の色文字列のいずれかです．RGB(A)配列は，0から1の値のみを含みます．例えば:

  • 'white'

  • 'w'

  • [1.0, 1.0, 1.0]

  • [0.5, 1.0, 0.7, 1.0]

  • '#FFFFFF'

opacityfloat, optional

返される色のデフォルトの不透明度です． color が長さ 4 の RGBA シーケンスでない場合に使用します．0 から 1 の間の不透明度のみが許可されます．

default_colorcolor_like, optional

color が None の場合に使用されるデフォルトの色です． もしこの値が None であれば，デフォルトではグローバルテーマカラーを使用します． フォーマットは color と同じです．

戻り値
tuple

opacity が設定されていない場合は長さ3のRGBシーケンス， opacity が設定されているか，入力の color がRGBAシーケンスの場合はRGBAシーケンスとなります．

例

>>> import pyvista
>>> pyvista.parse_color('blue')
(0.0, 0.0, 1.0)

>>> pyvista.parse_color('k')
(0.0, 0.0, 0.0)

>>> pyvista.parse_color('#FFFFFF')
(1.0, 1.0, 1.0)

>>> pyvista.parse_color((0.4, 0.3, 0.4, 1))
(0.4, 0.3, 0.4, 1.0)
