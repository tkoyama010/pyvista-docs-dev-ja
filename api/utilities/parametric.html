パラメトリックオブジェクト¶

これらのオブジェクトは，独立変数のセットでパラメータ化されたサーフェスを表します．これらの中には，Mobiusの帯のように，陰関数を使って（正しく，あるいは全く）表現することができないものもあります．

パラメトリック表面を作成するには，次の関数を使用します．その他の例を見るには， パラメトリックジオメトリオブジェクト を参照してください．

ParametricBohemianDome([a])

Bohemianドーム面を生成します．

ParametricBour(**kwargs)

Bourの最小表面を生成します．

ParametricBoy([zscale])

Boyの表面を生成します．

ParametricDini([a, b])

Diniの表面を生成します．

ParametricCatalanMinimal(**kwargs)

Catalanの最小表面を生成します．

ParametricConicSpiral([a, b, c, n])

貝殻のような円錐形の螺旋面を生成します．

ParametricCrossCap(**kwargs)

クロスキャップを生成します．

ParametricEllipsoid([xradius, yradius, zradius])

準拠楕円体を生成します．

ParametricEnneper(**kwargs)

Enneperの表面を生成します．

ParametricFigure8Klein([radius])

字形-8のKleinのボトルを作成します．

ParametricHenneberg(**kwargs)

Hennebergの最小サーフェスを生成します．

ParametricKlein(**kwargs)

Kleinボトルの "古典的な" 表現を生成します．

ParametricKuen([deltav0])

Kuensの表面を生成します．

ParametricMobius([radius])

Mobiusの帯を生成します．

ParametricPluckerConoid([n])

Pluckerの円錐表面を生成します．

ParametricPseudosphere(**kwargs)

pseudosphereを生成します．

ParametricRandomHills([numberofhills, ...])

ランダムに配置された丘で覆われた表面を生成します．

ParametricRoman([radius])

SteinerのRoman表面を生成します．

ParametricSuperEllipsoid([xradius, yradius, ...])

超楕円体を生成します．

ParametricSuperToroid([ringradius, ...])

スーパートロイドを生成します．

ParametricTorus([ringradius, crosssectionradius])

トーラスを生成します．

次の関数は，パラメトリックサーフェスの作成をサポートしています．

parametric_keywords(parametric_function[, ...])

パラメトリック関数にキーワード引数を適用します．

surface_from_para(parametric_function[, ...])

パラメトリック関数からメッシュを作成します．
