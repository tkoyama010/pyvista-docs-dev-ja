フィルタ¶

データセットフィルタ¶

pyvista.DataSetFilters は pyvista.DataSet によって継承され，以下のすべてのフィルタをPyVistaデータセットから直接呼び出し可能なメソッドとして利用できるようにします．

DataSetFilters(*args, **kwargs)

任意のvtkDataSetに適用できる共通フィルタのセット．

PolyDataフィルタ¶

pyvista.PolyDataFilters は pyvista.PolyData によって継承され，以下のすべてのフィルタを PolyData メッシュから直接呼び出し可能なメソッドとして利用できるようにします．

PolyDataFilters(*args, **kwargs)

ポリデータデータセットのフィルタ/アルゴリズムを管理する内部クラス．

UnstructuredGrid フィルタ¶

pyvista.UnstructuredGridFilters は pyvista.UnstructuredGrid によって継承され，以下のすべてのフィルタを UnstructuredGrid メッシュから直接呼び出し可能なメソッドとして利用できるようにします．

UnstructuredGridFilters(*args, **kwargs)

非構造化グリッドデータセットのフィルタ/アルゴリズムを管理する内部クラス．

UniformGridフィルタ¶

pyvista.UniformGridFilters は pyvista.UniformGrid によって継承され，以下のすべてのフィルタを UniformGrid メッシュから直接呼び出し可能なメソッドとして利用できるようにします．

UniformGridFilters(*args, **kwargs)

構造化グリッドデータセットのフィルタ/アルゴリズムを管理する内部クラス．

Composite フィルタ¶

これらは複合データセット，すなわち pyvista.MultiBlock に適用できるフィルタです． pyvista.CompositeFilters クラスは， pyvista.DataSetFilters から多くのフィルタを継承しますが，すべてを継承するわけではありません．

CompositeFilters(*args, **kwargs)

複合データデータセットのフィルタ/アルゴリズムを管理する内部クラス．
