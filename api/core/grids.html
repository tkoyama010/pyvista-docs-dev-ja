グリッドデータ¶

PyVistaのグリッドデータとは、位相的に規則的な点の間隔を持つデータセットのことです。 これらは ポイントセット よりも柔軟性に欠けますが、データセットのポイントやジオメトリを明示的に記述する必要がなく、わずかなパラメータで記述できるため、メモリ効率が非常に高くなります。

PyVistaのグリッドデータは、 pyvista.RectilinearGrid と pyvista.UniformGrid クラスで構成されています。 これらのクラスは vtkRectilinearGrid と vtkImageData クラスを継承しており、画像やボリュームデータのモデリングによく使われます。

pyvista.RectilinearGrid は、3つの座標方向に可変の間隔を持つデータセットをモデル化するために使用されます。

pyvista.UniformGrid は、3つの座標方向に一定の間隔を持つデータセットをモデル化するために使用されます。

クラスの説明

以下の表は、PyVistaのグリッドセットクラスの説明です。 これらのクラスは、対応する VTK の vtkRectilinearGrid, vtkImageData のスーパークラスからすべてのメソッドを継承しています。

3つの座標方向に可変の間隔を持つデータセット。

3つの座標方向に均一な間隔でデータセットをモデル化します。
