boolean_intersection#

PolyDataFilters.boolean_intersection(other_mesh, tolerance=1e-05, progress_bar=False)[ソース]#

2つのメッシュにブーリアンの交差演算を行います．

基本的に，ブーリアン結合，切断，交差はすべて同じ操作です．ただ，オブジェクトの異なる部分が最終的に保持されるだけです．

2つの多様体メッシュ``A``と``B```の交点は，``A``の体積が``B``の中にもあるメッシュである．

注釈

ブーリアン演算が思った通りに反応しない(例えば，間違った部分が消えてしまう)場合，メッシュの1つが法線を内側に向けている可能性があります．法線を視覚化するには， PolyDataFilters.plot_normals() を使用してください．

注釈

PolyDataFilters.intersection() フィルタは2つのメッシュの表面上の交点を返すのに対し，このメソッドは2つのメッシュ間の "volume" の交点を返します（これはしばしば線が解となります）．

注釈

両方のメッシュはすべての三角形で構成されていなければなりません． これは PolyData.is_all_triangles でチェックし， PolyDataFilters.triangulate() で変換します．

バージョン 0.32.0 で追加.

パラメータ
other_meshpyvista.PolyData

ソースのメッシュを操作するメッシュ．

tolerancefloat, optional

点の絶対的な距離がゼロとみなされるタイミングを決定するために使用される許容範囲．

progress_barbool, optional

進行状況を示す進行状況バーを表示します．

戻り値
pyvista.PolyData

ブーリアン演算の結果です．

例

2つの球体を使ってブーリアン交叉を実演します． 最終的なメッシュに2つの交叉のみ含まれていることに注目してください．

>>> import pyvista
>>> sphere_a = pyvista.Sphere()
>>> sphere_b = pyvista.Sphere(center=(0.5, 0, 0))
>>> result = sphere_a.boolean_intersection(sphere_b)
>>> pl = pyvista.Plotter()
>>> _ = pl.add_mesh(sphere_a, color='r', style='wireframe', line_width=3)
>>> _ = pl.add_mesh(sphere_b, color='b', style='wireframe', line_width=3)
>>> _ = pl.add_mesh(result, color='tan')
>>> pl.camera_position = 'xz'
>>> pl.show()

このフィルターを使ったその他の例については， ブール演算 を参照してください．
