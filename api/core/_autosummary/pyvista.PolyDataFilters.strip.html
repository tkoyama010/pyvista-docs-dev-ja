strip¶

PolyDataFilters.strip(join=False, max_length=1000, pass_cell_data=False, pass_cell_ids=False, pass_point_ids=False, progress_bar=False)[ソース]¶

ポリゴンデータセルをストリップします．

入力ポリゴン，三角形ストリップ，およびラインから三角形ストリップまたはポリラインを生成します．

ポリゴンは，三角形である場合にのみ三角形ストリップにアセンブルされます．他のタイプのポリゴンは出力に渡され，ストリップされません．(すべてのデータを除去する必要がある場合は，このフィルタを実行する前に， triangulate フィルタを使用して三角形以外のポリゴンを三角形化します．)フィルタは，入力ポリデータに(出力に)頂点が存在する場合は通過します．

また，入力で三角形のストリップまたはポリラインが定義されている場合，それらはパススルーされ，結合も延長もされません．(これらを除去したい場合は，このフィルタを実行する前に， triangulate フィルタを使って入力を三角形と線に分割してください．)

パラメータ
joinbool, optional

True の場合，出力ポリゴンセグメントが連続していれば結合されます．これは，サーフェスをスライスした後に便利です．デフォルトは False です．

max_lengthint, optional

三角形ストリップ内の三角形の最大数，およびポリライン内のラインの最大数を指定します．

pass_cell_databool, optional

入力のCellDataをFieldDataとして出力に渡すことを有効/無効にします．フィールドデータが変換されます．デフォルトは False です．

pass_cell_idsbool, optional

True の場合，出力ポリゴンデータセットには，各出力セルを生成したオリジナルの3 Dセルのセルインデックスを保持するcelldata配列が含まれます．これはピッキングに便利です．デフォルトでは，メモリを節約するために False になっています．

pass_point_idsbool, optional

True の場合，出力ポリゴンデータセットには，各出力頂点を生成したオリジナルの頂点のポイントインデックスを保持するポイントデータ配列が含まれます．これはピッキングに便利です．デフォルトでは，メモリを節約するために False になっています．

progress_barbool, optional

進行状況を示す進行状況バーを表示します．

戻り値
pyvista.PolyData

ストライプ状のメッシュ．

例

>>> from pyvista import examples
>>> mesh = examples.load_airplane()
>>> slc = mesh.slice(normal='z', origin=(0, 0, -10))
>>> stripped = slc.strip()
>>> stripped.n_cells
1
>>> stripped.plot(show_edges=True, line_width=3)
