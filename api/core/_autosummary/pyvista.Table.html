pyvista.Table¶

class Table(*args, **kwargs)¶

vtkTable クラスのラッパーです．

形状( n_columns の n_rows )の2 D NumPy配列を渡すか，NumPy配列を含むディクショナリから作成します．

例

>>> import pyvista as pv
>>> import numpy as np
>>> arrays = np.random.rand(100, 3)
>>> table = pv.Table(arrays)
