position¶

property Light.position¶

ライトの位置を取得します．

注: 位置は，ライトの(存在する場合)変換行列で示される座標空間で定義されます．ライトのワールドスペースの位置を取得するには， (読み込み専用) world_position プロパティを使用します．

例

初期化後に (10, 10, 10) に配置されるライトを作成し，その位置が重要な変換行列の影響を受けないことに注意してください．

>>> import numpy as np
>>> import pyvista as pv
>>> light = pv.Light()
>>> light.position = (10, 10, 10)
>>> # set a "random" transformation matrix
>>> light.transform_matrix = np.arange(4 * 4).reshape(4, 4)
>>> light.position
(10.0, 10.0, 10.0)
