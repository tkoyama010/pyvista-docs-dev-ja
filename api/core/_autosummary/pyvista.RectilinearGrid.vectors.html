vectors¶

property RectilinearGrid.vectors: Optional[pyvista.core.pyvista_ndarray.pyvista_ndarray]¶

アクティブなベクトルを取得します．

バージョン 0.32.0 で非推奨: ベクトルデータを返すための DataSet.vectors の使用は非推奨です．
