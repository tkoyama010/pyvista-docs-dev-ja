get_data_range¶

Table.get_data_range(arr=None, preference='row')¶

指定した配列のNaN以外の最小値と最大値を取得します．

パラメータ
arrstr, numpy.ndarray, optional

範囲を取得する配列の名前を指定します． None の場合，アクティブなスカラーが使用されます．

preferencestr, optional

スカラーが指定されている場合，これはデータセット内で検索するために推奨される配列型です． 'row' または 'field' のいずれかでなければなりません

戻り値
tuple

配列の (min, max) ．
