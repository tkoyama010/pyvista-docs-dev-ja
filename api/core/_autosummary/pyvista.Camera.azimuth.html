azimuth¶

property Camera.azimuth¶

カメラの方位角．

焦点を中心としたビューアップベクトルを中心にカメラを回転します．ビューアップベクトルはSetViewUpで設定されたものであり，投影方向に対して垂直である必要はありません．

例

>>> import pyvista
>>> pl = pyvista.Plotter()
>>> pl.camera.azimuth
0.0
>>> pl.camera.azimuth = 45.0
>>> pl.camera.azimuth
45.0
