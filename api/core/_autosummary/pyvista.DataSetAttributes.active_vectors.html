active_vectors#

property DataSetAttributes.active_vectors: Optional[numpy.ndarray]#

アクティブなベクトルをpyvista_ndarrayとして返します．

バージョン 0.32.0 で変更: アクティブなベクトルを設定するために使用することはできなりました． func:DataSetAttributes.set_vectors を使用するか，配列が既に存在している場合には DataSetAttribute.active_vectors_name に代入します．

例

点データを単純な立方体メッシュに関連付け，点配列のアクティブベクトルが最も最近に追加された配列であることを示します．

>>> import pyvista
>>> import numpy as np
>>> mesh = pyvista.Cube()
>>> vectors = np.random.random((mesh.n_points, 3))
>>> mesh.point_data.set_vectors(vectors, 'my-vectors')
>>> vectors_out = mesh.point_data.active_vectors
>>> vectors_out.shape
(8, 3)
