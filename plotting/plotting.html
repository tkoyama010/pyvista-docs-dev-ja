Jupyterノートブックでのプロット¶

静的および対話型インラインプロットは，Jupyterノートブックを使用して行うことができます．以下のコードスニペットは，レンダリングの静的スクリーンショットを作成し，Jupyterノートブックに表示します:

import pyvista as pv
sphere = pv.Sphere()
sphere.plot(jupyter_backend='static')

Plotter クラスを使用することもできます．

plotter = pv.Plotter(notebook=True)
plotter.add_mesh(sphere)
plotter.show(jupyter_backend='static')

さらに， panel や ipygany のようなJupyter plottingバックエンドを利用して，インタラクティブなプロットを生成することができます．オンラインでインタラクティブなドキュメントを作成するために使用することもできます．

plotter = pv.Plotter(notebook=True)
plotter.add_mesh(sphere)
plotter.show(jupyter_backend='ipygany')

詳細については， Jupyterノートブックプロット の項を参照してください．

背景プロッティング¶

PyVistaにはプロッタが用意されており，ユーザはバックグラウンドでレンダリングウィンドウを作成できます．このウィンドウは，ユーザが処理を実行している間もインタラクティブなままです．これにより，レンダリングシーンを作成し，シーンに対してインタラクティブにデータセットを追加または削除する機能や，一般的なシーン操作や書き出しタスクに便利なメニュー機能が作成されます．始めるために， pyvistaqt.BackgroundPlotter のインスタンス化を試してみてください:

import pyvista as pv
import pyvistaqt as pvqt
from pyvista import examples

dataset = examples.load_hexbeam()

p = pvqt.BackgroundPlotter()

p.add_mesh(dataset)

p.show_bounds(grid=True, location='back')

時系列データのプロット¶

次の例では，時間の経過とともに空間参照とデータ値が変化するデータをプロットする方法の概要を示します:

from threading import Thread
import time
import numpy as np
import pyvista as pv
import pyvistaqt as pvqt
from pyvista import examples


globe = examples.load_globe()
globe.point_arrays['scalars'] = np.random.rand(globe.n_points)
globe.set_active_scalars('scalars')


plotter = pvqt.BackgroundPlotter()
plotter.add_mesh(globe, lighting=False, show_edges=True, texture=True, scalars='scalars')
plotter.view_isometric()

# shrink globe in the background
def shrink():
    for i in range(50):
        globe.points *= 0.95
        # Update scalars
        globe.point_arrays['scalars'] = np.random.rand(globe.n_points)
        time.sleep(0.5)

thread = Thread(target=shrink)
thread.start()

便利な機能¶

pyvista.plot(var_item, off_screen=None, full_screen=False, screenshot=None, interactive=True, cpos=None, window_size=None, show_bounds=False, show_axes=None, notebook=None, background=None, text='', return_img=False, eye_dome_lighting=False, volume=False, parallel_projection=False, use_ipyvtk=None, jupyter_backend=None, return_viewer=False, jupyter_kwargs={}, theme=None, **kwargs)¶

vtkまたはnumpyオブジェクトをプロットします．

パラメータ

  • item (vtk or numpy object) -- プロットするVTKオブジェクトまたは numpy 配列．

  • off_screen (bool) -- True の場合，画面からプロットします．ウィンドウをポップアップしないでスクリーンショットを保存するのに便利です．

  • full_screen (bool, optional) -- ウィンドウを全画面で開きます．有効な場合， window_size を無視します．デフォルトは False です．

  • screenshot (str or bool, optional) -- 有効にすると，スクリーンショットをファイルに保存します．関連項目: help(pyvista.Plotter.screenshot) ．デフォルトの False ． True はスクリーンショットを取得し，イメージの numpy 配列を戻します．

  • window_size (list, optional) -- Window size in pixels. Defaults to [1024, 768]

  • show_bounds (bool, optional) -- True の場合，メッシュ境界を表示します．デフォルトは False です．エイリアス show_grid も使用できます．

  • notebook (bool, optional) -- True の場合，作成されたプロットはjupyterノートブック内に配置されます．jupyterコンソールがアクティブであると仮定します．

  • show_axes (bool, optional) -- vtk軸ウィジェットを表示します． None の場合， pyvista.global_theme.axes.show に従って有効になります．

  • text (str, optional) -- プロットの下部にテキストを追加します．

  • volume (bool, optional) -- ボリュームレンダリングには add_volume 方式を使用してください．

  • use_ipyvtk (bool, optional) -- 非推奨．代わりに， pyvista.set_jupyter_backend('ipyvtklink') または backend='ipyvtklink' を使用してグローバルにバックエンドを設定します．

  • jupyter_backend (str, optional) -- Jupyterノートブックが使用するバックエンドをプロットしています．次のいずれか: * 'none' : ノートブックに表示されません． * 'static' : 静止画を表示します． * 'panel' : ipygany ウィジェットを表示します * 'panel' : panel ウィジェットを表示します．これは pyvista.set_jupyter_backend でグローバルに設定することもできます．

  • jupyter_kwargs (dict, optional) -- Jupyterノートブックプロットバックエンドのキーワード引数．

  • theme (pyvista.themes.DefaultTheme, optional) -- Plot固有のテーマ．

  • **kwargs (optional keyword arguments) -- その他のオプションについては， ヘルプ (pyvista.Plotter.add_mesh) を参照してください．

戻り値

  • cpos (list) -- カメラ位置，焦点，およびビューアップのリスト．

  • img (numpy.ndarray) -- ピクセルRGBと，オプションでアルファ値を含む配列．サイズ:

    • [ウィンドウの高さxウィンドウの幅x3]テーマが transparent_background=False に設定されている場合．

    • [ウィンドウの高さxウィンドウの幅x4]テーマが transparent_background=True に設定されている場合．

    screenshot=True の場合にのみ返されます．

サンプル

単純な球のエッジを表示しながらプロットします．

>>> import pyvista
>>> mesh = pyvista.Sphere()
>>> mesh.plot(show_edges=True)
pyvista.plot_arrows(cent, direction, **kwargs)¶

矢印をベクトルとしてプロットします．

パラメータ

  • cent (np.ndarray) -- 単一の3 Dポイントまたは3 Dポイントの配列を受け入れます．

  • directions (np.ndarray) -- 単一の3 D点または3 Dベクトルの配列を受け入れます．centと同じ数のアイテムを含める必要があります．

  • **kwargs (additional arguments, optional) -- ヘルプ(pyvista.plot) を参照

戻り値

戻り値の型

デプスのピーリングを有効にすると，半透明のジオメトリのレンダリングが向上します．

パラメータ

  • number_of_peels (int) -- 剥離層の最大数．初期値は4で， pyvista.global_theme に設定されます．特別な値0は最大制限がないことを意味します．正の値でなければなりません．

境界軸を追加します．

meshが指定されていない場合，最新の入力メッシュの境界を表示します．

パラメータ

  • mesh (pyvista.DataSet or pyvista.MultiBlock) -- 境界軸を描画する入力メッシュ．

  • bounds (list or tuple, optional) -- メッシュ境界をオーバーライドする境界． [xmin, xmax, ymin, ymax, zmin, zmax]

  • show_xaxis (bool, optional) -- x軸を表示します．デフォルトは True です．

  • show_yaxis (bool, optional) -- y軸を表示します．デフォルトは True です．

  • show_zaxis (bool, optional) -- z軸を表示します．デフォルトは True です．

  • show_xlabels (bool, optional) -- xラベルを表示します．デフォルトは True です．

  • show_ylabels (bool, optional) -- yラベルを表示します．デフォルトは True です．

  • show_zlabels (bool, optional) -- zラベルを表示します．デフォルトは True です．

  • italic (bool, optional) -- 軸ラベルと数値を斜体にします．デフォルトは False です．

  • bold (bool, optional) -- 軸ラベルと数値を太字にします．デフォルトは True です．

  • shadow (bool, optional) -- テキストに黒い影を追加します．デフォルトは False です．

  • font_size (float, optional) -- ラベルフォントのサイズを設定します．デフォルトは16です．

  • font_family (string, optional) -- フォントファミリ． 'courier' ， 'times' ，または 'arial' のいずれかである必要があります．

  • color (string or 3 item list, optional) -- すべてのラベルと軸ラベルの色です．デフォルトは白です．文字列，RGBリスト，または16進カラー文字列．例: * color='white' * color='w' * color=[1, 1, 1] * color='#FFFFFF'

  • xlabel (string, optional) -- X軸のタイトル．既定 "X Axis"

  • ylabel (string, optional) -- Y軸のタイトル．既定 "Y Axis"

  • zlabel (string, optional) -- Z軸のタイトル．既定 "Z Axis"

  • use_2d (bool, optional) -- Windowsのvtk 6.3のバグにより，この関数がクラッシュするようです．これを有効にすると，他の環境でのプロットがよりスムーズになります．
