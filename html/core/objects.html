オブジェクト¶

pyvista.DataObject クラスは，すべてのPyVista型に共通のメソッドと属性のセットです．これらのオブジェクトには空間参照はなく，単にデータが保持されます．

アトリビュート

actual_memory_size

データセットオブジェクトの実際のサイズを返します．

field_arrays

DataSetAttributesとしてvtkFieldDataを返します．

memory_address

基本となるC++オブジェクトのアドレスを 'Addr=%p' 形式で取得します．

メソッド

add_field_array(scalars, name[, deep])

フィールド配列を追加します．

clear_field_arrays()

すべてのフィールド配列を削除します．

copy([deep])

オブジェクトのコピーを返します．

copy_attributes(dataset)

入力データセットオブジェクトのデータ属性をコピーします．

copy_meta_from(ido)

別のオブジェクトからこのオブジェクトにpyvistaメタデータをコピーします．

copy_structure(dataset)

入力データセットオブジェクトの構造(ジオメトリとトポロジ)をコピーします．

deep_copy(to_copy)

このメッシュを指定されたメッシュで深いコピーとして上書きします．

get_data_range()

指定した配列のNaN以外の最小値と最大値を取得します．

このデータセットのヘッダ統計情報を返します．

このvtkオブジェクトをファイルに保存します．

shallow_copy(to_copy)

指定されたメッシュをこのメッシュに浅くコピーします．

property actual_memory_size¶

データセットオブジェクトの実際のサイズを返します．

戻り値

データセットオブジェクトの実際のサイズ(1024バイト)．

戻り値の型

int

サンプル

>>> from pyvista import examples
>>> mesh = examples.load_airplane()
>>> mesh.actual_memory_size  
93

オブジェクトのコピーを返します．

パラメータ

戻り値

戻り値の型

copy_attributes(dataset: vtkmodules.vtkCommonDataModel.vtkDataSet)¶

入力データセットオブジェクトのデータ属性をコピーします．

サンプル

>>> import pyvista as pv
>>> source = pv.UniformGrid((10, 10, 5))
>>> source = source.compute_cell_sizes()
>>> target = pv.UniformGrid((10, 10, 5))
>>> target.copy_attributes(source)
>>> cpos = target.plot(scalars='Volume', show_edges=True)

save(filename: str, binary=True)¶

このvtkオブジェクトをファイルに保存します．

パラメータ

  • filename (str, pathlib.Path) -- 出力ファイルのファイル名．Writerのタイプは，ファイル名の拡張子から推測されます．

  • binary (bool, optional) -- Trueの場合はバイナリ，それ以外の場合はASCIIで記述します．

メモ

バイナリファイルはASCIIよりもはるかに高速に書き込み，ファイルサイズも小さくなります

class pyvista.Table(*args, **kwargs)¶

ベースクラス: vtkmodules.vtkCommonDataModel.vtkTable, pyvista.core.dataobject.DataObject

vtkTable クラスのラッパーです．

形状( n_columns の n_rows )の2 D NumPy配列を渡すか，NumPy配列を含むディクショナリから作成します．

サンプル

>>> import pyvista as pv
>>> import numpy as np
>>> arrays = np.random.rand(100, 3)
>>> table = pv.Table(arrays)
get(index)¶

名前で配列を取得します．

get_data_range(arr=None, preference='row')¶

指定した配列のNaN以外の最小値と最大値を取得します．

パラメータ

  • arr (str, np.ndarray, optional) -- 範囲を取得する配列の名前を指定します．Noneの場合，アクティブなスカラーが使用されます．

  • preference (str, optional) -- スカラーが指定されている場合，これはデータセット内で検索するために推奨される配列型です． 'row' または 'field' のいずれかでなければなりません
