グリッドデータ¶

pyvista内のグリッド化されたデータセットは，VTKバックエンド上の vtkRectilinearGrid および vtkImageData データ・オブジェクトをキャプチャします．これらのデータ型には， pyvista.Grid の共有継承に含まれる共通の特徴があります．

アトリビュート

dimensions

グリッドの次元の長さ3タプルを戻します．

メソッド

ベースクラス: pyvista.core.dataset.DataSet

非ポイントセットグリッド用の共通メソッドでいっぱいのクラス．

property dimensions¶

グリッドの次元の長さ3タプルを戻します．

これらは，事実上，3つのデータセット軸のそれぞれに沿ったノードの数です．

class pyvista.RectilinearGrid(*args, **kwargs)¶

ベースクラス: vtkmodules.vtkCommonDataModel.vtkRectilinearGrid, pyvista.core.grid.Grid

vtk.vtkRectilinearGridオブジェクトの機能を拡張します．

いくつかの方法で初期化できます:

  • 空のグリッドを作成

  • vtk.vtkRectilinearGrid オブジェクトから初期化します

  • 点配列から直接初期化します

点配列からの初期化の詳細については，ドキュメントの_from_arraysを参照してください．

サンプル

>>> import pyvista
>>> import vtk
>>> import numpy as np

>>> # Create empty grid
>>> grid = pyvista.RectilinearGrid()

>>> # Initialize from a vtk.vtkRectilinearGrid object
>>> vtkgrid = vtk.vtkRectilinearGrid()
>>> grid = pyvista.RectilinearGrid(vtkgrid)

>>> # Create from NumPy arrays
>>> xrng = np.arange(-10, 10, 2)
>>> yrng = np.arange(-10, 10, 5)
>>> zrng = np.arange(-10, 10, 1)
>>> grid = pyvista.RectilinearGrid(xrng, yrng, zrng)

class pyvista.UniformGrid(*args, **kwargs)¶

ベースクラス: vtkmodules.vtkCommonDataModel.vtkImageData, pyvista.core.grid.Grid, pyvista.core.filters.UniformGridFilters

vtk.vtkImageDataオブジェクトの機能を拡張します．

いくつかの方法で初期化できます:

  • 空のグリッドを作成

  • vtk.vtkImageData オブジェクトから初期化します

  • 点配列から直接初期化します

点配列からの初期化の詳細については，ドキュメントの _from_specs を参照してください．

サンプル

>>> import pyvista
>>> import vtk
>>> import numpy as np

>>> # Create empty grid
>>> grid = pyvista.UniformGrid()

>>> # Initialize from a vtk.vtkImageData object
>>> vtkgrid = vtk.vtkImageData()
>>> grid = pyvista.UniformGrid(vtkgrid)

>>> # Using just the grid dimensions
>>> dims = (10, 10, 10)
>>> grid = pyvista.UniformGrid(dims)

>>> # Using dimensions and spacing
>>> spacing = (2, 1, 5)
>>> grid = pyvista.UniformGrid(dims, spacing)

>>> # Using dimensions, spacing, and an origin
>>> origin = (10, 35, 50)
>>> grid = pyvista.UniformGrid(dims, spacing, origin)
