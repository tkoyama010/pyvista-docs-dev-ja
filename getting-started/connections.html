接続#

PyVistaは強力なツールであり，研究者はこれを利用して，大規模なデータセットを直感的かつPython的な方法で，説得力のある統合された可視化を作成することができます．

PyVistaは、科学や工学のさまざまな分野で、データやモデルを3Dで視覚化し、出版品質の図版を生成し、解析ワークフローを自動化し、PyVistaの3D機能を活用したカスタムアプリケーションを構築するために、多様なコミュニティによって利用されています。

PyVistaを活用したオープンソースプロジェクトをいくつか紹介します。

エンジニアリング#

  • pyleecan: 電気機械や駆動装置の設計や最適化のための，ユーザーフレンドリーで統一された柔軟なシミュレーションフレームワークを提供する電気工学オープンソースソフトウェア．

  • pymapdl: Ansys MAPDLへのPythonicインターフェース．

  • pyprocar: 電子構造の前処理・後処理のためのPythonライブラリです．

  • AeroSandbox: 空気力学、推進力、構造、軌道設計などのためのプラグアンドプレイ解析ツール。

  • Ptera: 羽ばたき飛行を解析するためのソフトウェア。

地球科学#

  • GemPy: PyVistaを用いた3次元構造地質モデリングソフトウェアで、3次元の可視化とインタラクションが可能です。

  • gemgis: ジオモデリングのための空間データ処理．

  • gimli: 地球物理学のインバージョンとモデリングのためのライブラリです．

  • discretize: 有限体積問題および逆問題のための離散化ツール。

  • SEGY-SAK: SEGY Swiss Army Knife for Seismic Data.

  • subsurface: 地下科学と工学のためのコアデータ交換ライブラリ。

  • Flownet: データ駆動型貯水池予測。

  • PVGeo: 地球科学データとモデルを分析するためのVTKベースのアルゴリズムのPythonパッケージです．PyVistaは，PVGeoのアルゴリズムの入出力をよりアクセスしやすくするために使用されます．

  • omfvista: Open Mining Format (omf) の3 D可視化．PyVistaは，このライブラリーの視覚化の基礎を提供します．

その他#

  • VesselVio: セグメント化された血管系データセットの解析と可視化。

  • pymeshfix: 受賞歴のあるMarco Atteneの素晴らしい MeshFix ソフトウェアのPython/Cythonラッパーです．

  • tetgen: Hang SiのC++TetGenライブラリへのPythonインタフェース．

  • itkwidgets: イメージ，点セット，メッシュを2 Dおよび3 Dで視覚化するインタラクティブなJupyterウィジェット．すべてのPyVistaメッシュタイプをサポートします．

  • mne-python: MNE: 脳磁図(MEG)と脳波(EEG)をPythonで．

  • pyntcloud: 3 次元点群を扱うための Python ライブラリ。

  • DiSECt: 微分可能な切断シミュレータ。

  • torch-points3d: 点群に対して深層学習を行うための Pytorch フレームワーク。

  • pyprocar: 電子構造の前処理・後処理のためのPythonライブラリです．

  • FEniCSx: FEniCS PythonとC++による問題解決環境．
